# Chrome 浏览器集成

需通过 `claude --chrome` 启动。

## 允许的阶段

| 阶段 | Chrome | 用途 |
|------|--------|------|
| P2 | **允许** | UI/UX 调研、竞品分析、原型预览 |
| P4 | **允许** | 视觉回归测试、E2E 交互验证、响应式测试 |
| 其他 | 禁止 | — |

## P2 原型展示流程

涉及 UI 的任务必须在 P2 创建原型并在 Chrome 中展示给用户确认：

1. **创建原型 HTML** — 自包含文件（内联 CSS + JS），可引用 CDN 组件库
2. **Chrome 打开预览** — 本地打开让用户直观看到设计效果
3. **截图记录** — 截取关键页面作为设计参考
4. **迭代优化** — 根据用户反馈修改，再次预览，直到满意
5. **确认后进入 P3** — P3 编码以此原型为视觉基准

## P4 使用规范

- 打开本地开发服务器验证 UI
- 截图对比渲染结果
- 验证交互流程和响应式布局
