# Chrome 浏览器集成

Chrome 工具需通过 `claude --chrome` 启动。仅在以下阶段允许使用。

## 允许的阶段

| 阶段 | Chrome | 用途 |
|------|--------|------|
| P2 | **允许** | UI/UX 调研、竞品分析、设计参考截图 |
| P4 | **允许** | 视觉回归测试、E2E 交互验证、响应式测试 |
| 其他 | 禁止 | — |

## P2 使用规范
- 访问竞品和 UI 组件库文档
- 截图保存设计参考
- 禁止登录私人账户、提交表单、下载文件

## P4 使用规范
- 打开本地开发服务器验证 UI
- 截图对比渲染结果
- 验证交互流程和响应式布局
- 禁止访问外部网站
